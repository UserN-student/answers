**3. Для чего при разработке искусственного интеллекта используются базы данных?**  
> Алгоритмы ИИ извлекают информацию из больших объемов данных, для хранения которых нужны базы данных

**4. Для чего используется система Hadoop?**  
> Для параллельной обработки огромных баз данных

**5. Какой набор инструментов чаще всего используется в машинном обучении?**

**6. В чем преимущества использования систем на основе искусственного интеллекта в бизнесе?**

> - Дообучаемость: при появлении новых данных алгоритм можно дообучить на них, чтобы алгоритм выполнял правильные предсказания в более широком круге сценариев
